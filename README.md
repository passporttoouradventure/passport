# Passport to Our Adventure

GitHub Pages-ready wedding invitation for Isabella & Kevin.

## Upload
Upload **everything inside this folder** to the root of your GitHub repository. Do not upload only the ZIP file.

Then open **Settings → Pages**, choose **Deploy from a branch**, select **main** and **/(root)**, and save.

Expected address for a repository named `passport`:
https://passporttoouradventure.github.io/passport/

Music starts after a guest taps “Begin Your Journey,” as required by mobile browsers.
